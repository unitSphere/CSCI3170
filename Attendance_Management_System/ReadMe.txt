This is an Attendance Management System written in COBOL.
It takes 3 input files attendance.txt, employees.txt and monthlyattendance.txt and outputs the summery record for each employee in symmerycob.txt file.
It also simultaneously updates mothly attendance and writes the results in an output file called monthlyattendancecob.cob

Program file name: atd.cob
